# anh-co2-almacenamiento-teledeteccion
Proyecto de la ANH para evaluar la viabilidad del almacenamiento geológico de CO₂ en cuencas sedimentarias de Colombia, integrando datos satelitales (GRACE, InSAR, GLDAS), modelación numérica y algoritmos de aprendizaje asistido para reducir incertidumbres y analizar interacciones con aguas subterráneas.
